# 哎纸娃娃
- Andrid Studio项目
- 手机端的纸娃娃（游戏素材）生成工具

应该是安卓手机上第一个纸娃娃软件了！  
功能就是！可自行导入素材的纸娃娃合成软件！  
用法于现有的pc纸娃娃软件相似，和RPG游戏引擎内置的纸娃娃软件也一样！  
简单点击，能兼容网络上128*192像素对以及相同比缩放的4*4规格的纸娃娃行走图素材  
随意混搭 一键保存  
（需要sd卡读写权限）
